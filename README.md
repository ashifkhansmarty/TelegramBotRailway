\# Telegram Mobile Info Bot



\*\*How to deploy on Render.com:\*\*



1\. Fork or clone this repository.

2\. Set Environment Variable `BOT\_TOKEN` with your Telegram bot token.

3\. Deploy the repo to Render.com:

&nbsp;  - Environment: Python 3.13

&nbsp;  - Build Command: `pip install -r requirements.txt`

&nbsp;  - Start Command: `python bot.py`

4\. Keep bot alive using UptimeRobot pinging the Render URL.
